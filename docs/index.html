About RegressionX

RegressionX merupakan aplikasi berbasis web yang dirancang secara khusus untuk membantu mahasiswa, pendidik, dan praktisi data dalam melakukan analisis Regresi Linear sederhana secara cepat, efisien, dan interaktif.

Selain menghitung nilai intercept ($a$) dan slope ($b$), sistem juga secara dinamis merumuskan korelasi ($r$), determinasi ($R^2$), serta RMSE untuk mengukur tingkat akurasi model regresi terhadap data Anda.
